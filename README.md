# 寒仔的学术笔记

这是一个基于 **Hugo + Stack 主题** 的个人学术博客项目，用于长期记录与整理我在经济学研究中的思考、模型推导与方法笔记。

---

## 项目定位

本项目不是传统意义上的“个人博客”，而是一个 **学术研究笔记系统**，主要用途包括：

- 记录理论模型与推导过程
- 整理研究中的中间想法与失败尝试
- 构建可长期演化的研究知识库
- 为 future working paper / dissertation 提供素材沉淀

---

## 技术架构概览

整体架构遵循 **“本地写作 + 静态编译 + 服务器托管”** 的设计原则：

### 核心组件

- **Hugo**：静态站点生成器
- **Stack**：Hugo 主题（支持数学公式、目录、标签、多语言）
- **KaTeX**：数学公式渲染
- **Nginx**：文件服务器
- **GitHub**：源码版本控制与备份

---

## 目录结构说明

```text
.
├── content/            # 博客正文（Markdown 原稿）
│   └── posts/          # 学术笔记文章
│   └── publications/   # 论文发表记录
├── assets/             # 主题资源（头像、样式等）
├── static/             # 静态资源（不经 Hugo 处理）
├── themes/             # Hugo 主题（Stack）
├── config.toml         # Hugo 全局配置
├── archetypes/         # 文章模板
├── README.md           # 项目说明
└── .gitignore
```
